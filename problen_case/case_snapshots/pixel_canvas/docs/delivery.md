# Pixel Canvas - 传送带清板益智游戏

## 项目概述

**Pixel Canvas** 是一款像素风格的传送带清板益智游戏，玩家通过点击派出不同颜色的小猪进入传送带，利用位置依赖的索敌机制和颜色匹配攻击，逐步清空中央棋盘上的目标块。

## 运行方式

双击打开 `game/index.html` 即可开始游戏，无需服务器。

## 核心玩法

1. **四周环形传送带**：小猪沿棋盘外圈持续移动
2. **位置依赖索敌**：小猪只能攻击当前位置正对方向的第一个目标块
3. **颜色匹配攻击**：只有小猪颜色与目标块颜色相同时才攻击
4. **等待槽回收**：未耗尽的小猪返回等待槽，可再次派出
5. **多关卡递进**：3个关卡，颜色种类递增（2色→3色→4色）

## 操作说明

- 点击底部等待槽中的小猪派出到传送带
- 小猪自动沿传送带移动并攻击同色目标块
- 清空所有目标块即可通关

## 技术实现

- **引擎**：Canvas 2D
- **运行模式**：file（单文件，双击打开）
- **视觉风格**：像素复古风

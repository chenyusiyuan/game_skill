## 游戏简介

像素风、轻度休闲、益智策略、竖屏点击游戏。玩家点击不同颜色的小猪进入棋盘四周的传送带，小猪沿外圈移动，只能根据当前位置依序处理正对方向上的目标块。通过合理安排出场顺序、颜色搭配和等待槽管理，逐步清空整块像素区域。

## 核心玩法

### 必须实现功能

1. **四周环形传送带** - 小猪沿外圈持续移动
2. **颜色匹配攻击** - 只攻击正对方向第一个同色目标块
3. **小猪单位系统** - 颜色属性 + 弹药数值
4. **等待槽回收** - 未耗尽小猪可回收再派出
5. **中央棋盘目标块** - 普通块、强化块、空白格
6. **位置依赖索敌** - 严格按当前位置处理目标
7. **胜负条件** - 清空棋盘=胜利，资源耗尽=失败

### 硬约束

- ✅ 禁止全屏自动索敌
- ✅ 禁止多目标攻击
- ✅ 禁止穿透攻击
